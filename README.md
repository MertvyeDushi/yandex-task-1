Для верстки используется шаблонизатор Pug:
классы через точку по БЭМ (насколько я знаю БЭМ), классы в скобках - утилитарные классы.

Для Vote шаблона дизайн задан таким образом, что невозможно изменить положение элементов,
изменив всего лишь несколько свойств в стилях, поэтому было принято решение в некоторых
местах использовать класс horizontal для скрытия некоторых элементов со страницы при
альбомной ориентации и класс vertical - при портретной.

В шаблоне Activity высота и ширина ячеек адаптивны.

Diagram template: закругления на углах можно задать кубическими кривыми Безье, но в данном
случае для упрощения диаграмма построена с помощью эллиптических кривых.

В некоторых шаблонах необходимо было использовать циклы для заполнения данных. К сожалению,
задание было не до конца ясно до последнего дня и времени переделывать уже не осталось.

Некоторые шаблоны не компилировались из ./src/templates/*.pug, а изменялись непосредственно в stories.js.

Для шаблона Activity не использовался проход по свойствам объекта из-за отсутствия порядка свойств.
